# Budgetly 💰

A personal budget tracking mobile app built with React Native, Expo & TypeScript.

## Features
- Add income and expense transactions
- Categorize transactions (Salary, Food, Transport, etc.)
- View total income, expenses and balance
- Transaction history with filter
- Data saved locally on device
- Light and dark theme

## Tech Stack
- React Native
- Expo
- TypeScript
- AsyncStorage
